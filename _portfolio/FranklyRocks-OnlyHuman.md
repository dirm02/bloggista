# üåç OnlyHuman

Tired of seeing AI-generated SEO spam choke your search results?

**OnlyHuman** is a filter list for uBlock Origin that blocks low-effort AI content farms and SEO spam websites from appearing in your search results and feeds.

## üéØ What does it block?

OnlyHuman specifically targets:

- **AI-SEO content farms**,  websites designed to rank for search queries by mass-producing low-effort AI rewrites.
- **Thin SEO sites**,  pages with thousands of words saying nothing, purely optimized for search algorithms.
- **Spun content and rewrites**,  identical articles regurgitated across multiple domains.
- **Search result spam**,  low-quality sites that flood **Google**, **Bing**, **DuckDuckGo**, **Yandex** with filler content.

If it's a website built purely to exploit search engines with AI-generated fluff, it gets blocked.

## ‚öôÔ∏è How to install

1. Install [uBlock Origin](https://github.com/gorhill/uBlock).
2. Go to "Filter lists" ‚Üí "Import" and paste this URL:
```
https://raw.githubusercontent.com/FranklyRocks/OnlyHuman/refs/heads/main/list.txt
```
3. Apply changes.
4. Enjoy cleaner search results.

## ‚úÖ What we don't block

- Legitimate AI tools and services.
- Websites using AI as part of a broader offering.
- Real content, even if imperfect.

This is a **junk filter for SEO spam**, not a censorship tool.

## üì£ Want to contribute?

You can:

- Suggest AI-SEO sites to block (via [Issues](https://github.com/FranklyRocks/onlyhuman/issues))
- Submit domains and patterns
- Help maintain the list

## üß™ Disclaimer

OnlyHuman targets obvious AI-generated content farms. It won't catch everything, and it may occasionally block legitimate sites. The web is messy, so we do our best.

Your time is too valuable to waste on the hundredth AI rewrite of "productivity tips from famous dead people".
