## üìå Why Choose Palmr.?

- **Self-hosted** ‚Äì Deploy on your own server or VPS.
- **Full control** ‚Äì No third-party dependencies, ensuring privacy and security.
- **No artificial limits** ‚Äì Share files without hidden restrictions or fees.
- **Folder organization** ‚Äì Create folders to organize and share files.
- **Simple deployment** ‚Äì SQLite database and filesystem storage for easy setup.
- **Scalable storage** ‚Äì Optional S3-compatible object storage for enterprise needs.

## üöÄ Technologies Used

### **Palmr.** is built with a focus on **performance**, **scalability**, and **security**.

<div align="center">
  <img src="https://res.cloudinary.com/technical-intelligence/image/upload/v1745548231/Palmr./Captura_de_Tela_2025-04-24_a%CC%80s_23.24.26_kr4hsl.png" style="width: 100%; border-radius: 15px;" />
</div>


### **Backend & API**
- **Fastify (Node.js)** ‚Äì High-performance API framework with built-in schema validation.
- **SQLite** ‚Äì Lightweight, reliable database with zero-configuration setup.
- **Filesystem Storage** ‚Äì Direct file storage with optional S3-compatible object storage.

### **Frontend**
- **NextJS 15 + TypeScript + Shadcn/ui** ‚Äì Modern and fast web interface.


## üõ†Ô∏è How It Works

1. **Web Interface** ‚Üí Built with Next, React and TypeScript for a seamless user experience.
2. **Backend API** ‚Üí Fastify handles requests and manages file operations.
3. **Database** ‚Üí SQLite stores metadata and transactional data with zero configuration.
4. **Storage** ‚Üí Filesystem storage ensures reliable file storage with optional S3-compatible object storage for scalability.
